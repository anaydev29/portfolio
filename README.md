# Interactive 3D Developer Portfolio

Welcome to the repository for my interactive 3D developer portfolio. This project showcases my skills, projects, and experience as a developer using cutting-edge web technologies to create a modern and visually engaging user experience. 

Built and designed by **Anay Shivhare**.

### 🌟 Live Demo
Check out the live deployed version here: [https://portfolio-six-sandy-13.vercel.app/](https://portfolio-six-sandy-13.vercel.app/)

## ✨ Features

- **Immersive 3D Graphics**: Integrates Three.js and React Three Fiber to render dynamic backgrounds and floating geometries.
- **Smooth Animations**: Utilizes Framer Motion for sleek page transitions and element animations.
- **Responsive Layout**: Designed with Tailwind CSS to ensure a beautiful experience on both mobile and desktop screens.
- **Modern UI Components**: Features custom-built components like a dynamic Navbar, interactive Hero Scene, and stylized project cards.
- **Optimized Performance**: Built on top of Vite and React 19 for lightning-fast build times and seamless user interactions.

## 🛠️ Tech Stack

- **Framework:** React 19
- **Build Tool:** Vite
- **Styling:** Tailwind CSS (v4)
- **3D Rendering:** Three.js, React Three Fiber, React Three Drei
- **Animations:** Framer Motion
- **Icons:** Lucide React

## 🚀 Getting Started

To run this project locally, simply follow these steps:

### Prerequisites

Ensure you have [Node.js](https://nodejs.org/) installed on your machine.

### Installation

1. **Clone the repository:**
   ```bash
   git clone <your-repo-link>
   cd portfolio
   ```

2. **Navigate to the frontend directory:**
   ```bash
   cd frontend
   ```

3. **Install the dependencies:**
   ```bash
   npm install
   ```

4. **Start the development server:**
   ```bash
   npm run dev
   ```

5. Open [http://localhost:5173](http://localhost:5173) in your browser to view the portfolio.
